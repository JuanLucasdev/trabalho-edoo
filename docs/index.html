Sistema de Monitoramento Hospitalar

Sistema desenvolvido para a disciplina de Estruturas e Desenvolvimento Orientado a Objetos (EDOO)

Autores: Juan Lucas, Matheus Silva e William Santos

Sobre o Projeto

O Sistema de Monitoramento Hospitalar é uma aplicação web que simula operações básicas de um hospital, implementando cadastro de pacientes e médicos, gerenciamento de leitos, internação, alta e exportação de dados.

Desenvolvido com backend em C++ seguindo princípios de orientação a objetos e frontend em React para uma interface interativa.

Funcionalidades

Cadastro

Cadastro de pacientes e médicos com validação de dados

Gerenciamento de Leitos

Controle completo de ocupação e liberação de leitos hospitalares

Internação e Alta

Processo completo de internação e liberação de pacientes

Exportação de Dados

Geração de relatórios em formato JSON para análise

Arquitetura

Frontend (React)

Interface do usuário responsiva desenvolvida com React

Backend (C++)

API HTTP e lógica de negócio implementada em C++

Domínio (OO)

Classes Hospital, Paciente, Medico e Leito com princípios OO
